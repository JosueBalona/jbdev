Sobre o projeto

Este projeto foi desenvolvido a pedido de um cliente com o objetivo de criar uma plataforma de sustentabilidade, substituindo os processos anteriormente realizados em Excel. A nova solução visa otimizar o registo, a gestão e o acesso a informações relevantes ao longo de todo o ano.
A plataforma permite que os utilizadores insiram dados de forma contínua e organizada, garantindo que as informações estão sempre atualizadas. No final do ano, ou sempre que necessário, é possível gerar relatórios detalhados que consolidam os dados recolhidos, proporcionando uma visão completa e precisa das métricas de sustentabilidade. Adicionalmente, a plataforma foi concebida para facilitar auditorias, permitindo acesso rápido e confiável às informações armazenadas.
Esta solução centralizada elimina os desafios associados ao uso de folhas de cálculo, como dados desatualizados ou inconsistentes, ao mesmo tempo que melhora a eficiência, a precisão e a acessibilidade das informações. Assim, o cliente passa a contar com uma ferramenta robusta que apoia a tomada de decisões e reforça o compromisso com práticas sustentáveis.

Tecnologias utilizadas

ASP.NET | C# | SQL | CSS | JavaScript | Entity Framework 8.0 | HTML
