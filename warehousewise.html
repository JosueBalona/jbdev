WarehouseWise

Sobre o projeto

Este projeto foi elaborado no âmbito da unidade curricular de Projeto de Engenharia Informática em Contexto Empresarial, como parte integrante da licenciatura de Engenharia Informática e apresenta o desenvolvimento de uma Plataforma de Gestão de Armazéns.
Esta plataforma, permite efetuar toda a gestão, submissão e acompanhamento dos pedidos de material que sejam realizados pelos utilizadores.
Este processo vem colmatar processos que em várias empresas de produção que trabalhei no passado era realizado forma manual e que pode levar a perda de informação, erros no controlo de stock e erros na entrega de material. Este funcionamento, limita também a obtenção de relatório estatísticos.
Com esta plataforma, todo este processo ficará central, acessível aos utilizadores intervenientes no pedido, tornando toda a sua gestão e consulta fácil e intuitiva.

Tecnologias utilizadas

ASP.NET | C# | SQL | CSS | JavaScript | Entity Framework 8.0 | HTML
