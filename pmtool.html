Sobre o projeto

Este projeto foi desenvolvido a pedido de um cliente no âmbito de uma necessidade específica de gestão e acompanhamento de projetos, com o objetivo de otimizar o controlo e a execução de tarefas associadas a planos de projeto predefinidos.
A plataforma permite a gestão integral dos projetos, incluindo o acompanhamento do progresso através de relatórios detalhados, identificação de tarefas em atraso e verificação do seu encerramento atempado. Além disso, oferece funcionalidades adicionais indispensáveis, como a possibilidade de saltar tarefas que não são necessárias para um projeto específico, bem como a atribuição dinâmica de tarefas a utilizadores de acordo com as suas responsabilidades.
Este sistema centralizado vem substituir métodos manuais e dispersos que frequentemente levam a dificuldades de organização, falhas de comunicação e atrasos na execução. Com a implementação desta solução, todos os intervenientes terão acesso a uma visão clara do progresso, das responsabilidades e das prioridades, permitindo uma gestão mais eficiente, transparente e adaptada às necessidades reais do cliente.

Tecnologias utilizadas

ASP.NET | C# | SQL | CSS | JavaScript | Entity Framework 4.3| HTML
